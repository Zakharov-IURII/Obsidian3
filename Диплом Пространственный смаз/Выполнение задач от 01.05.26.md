1. Для считывания в файл используем функцию imread() библиотеки OpenCV (устанавливается через pip):
``` python
import cv2

import numpy as np

#считываем файл не напрямую imread - ошибки с кириллицей в пути - а через numpy

path = r'C:\Users\Юра\Desktop\Хранилище №3 (Work)\_Resourses\Изображение для демонстрации линейного горизонтального смаза.png'

#uint8 - беззнаковый int 8 бит (256 значений - как раз под уровни яркости)

img_array = np.fromfile(path, np.uint8)

img = cv2.imdecode(img_array, cv2.IMREAD_COLOR_BGR)

if img is None:

    print('Ошибка, файла нет')

else:

    print('Хорошо')

    #imshow ожидает данные для визуализации в окне именно в формате BGR и будет их так считывать, даже если

    #передать их ему в формате RGB - то есть он красный канал использует в качестве синего и т.п.

    #Поэтому img либо в BGR (флаг cv2.IMREAD_COLOR_BGR), либо нужно дополнительно переконвертировать массив

    cv2.imshow('Image', img)
    #собачка вывелась корректно

    cv2.waitKey(0)

    cv2.destroyAllWindows
```
