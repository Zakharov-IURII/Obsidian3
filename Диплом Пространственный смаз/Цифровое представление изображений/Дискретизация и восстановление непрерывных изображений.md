#### Основные понятия

Аналоговый сигнал: 
Cигнал существует в каждой точке;
Непрерывен по времени и амплитуде;
Вблизи любой точки с координатой x можно найти сколь угодно близкую соседнюю с координатой x + ▲x.

Дискретный сигнал: 
Cигнал существует только в отдельных точках k▲x по оси X, ▲x - расстояние м/у ними (часто фиксированное);
Непрерывная переменная x -> индекс k - дискретизация;
Значения в точках = отсчеты аналогового сигнала с неограниченной точностью (1.347368...) - непрерывен по амплитуде.

Цифровой сигнал:
Значения дискретного сигнала квантуют по уровням;
Квантование (в общем) - отображение множества непрерывных значений амплитуды в конечное дискретное множество уровней по заданным правилам
(Округление до ближайшего при равноудаленных границах - лишь один из простейших вариантов наравне с отбросом дробной части);

Кодирование:
Конечный набор уровней - каждому ставится в соответствие двоичное (к примеру) число;
Цифровой код - конечная размерность (разрядность), определяемая количеством уровней: 64 уровня - 6 бит для кодирования каждого значения (от 000000 до 111111).

Основные шаги перехода от аналогового сигнала к цифровому:
(Опционально) Предварительная фильтрация - Дискретизация - Квантование - (Опционально) Кодирование.
